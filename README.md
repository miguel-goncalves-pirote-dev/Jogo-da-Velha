# Jogo da Velha

Este é um projeto simples de **Jogo da Velha** desenvolvido em Python.  
O objetivo é praticar lógica de programação e uso de bibliotecas gráficas.

## Como executar

1. Certifique-se de ter o **Python 3** instalado.
2. Clone este repositório: **git clone https://github.com/miguel-goncalves-pirote-dev/jogo-da-velha.git**
3. Entre na pasta do projeto: cd **Jogo da Velha**
4. Execute o jogo: **python "Jogo da Velha.py"**

## Funcionalidades

- Dois jogadores (X e O)
- Verificação automática de vitória e empate
- Interface simples e intuitiva
